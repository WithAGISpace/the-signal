# Code of Conduct

## Our Standard

Be productive. Be respectful. Ship improvements.

This project exists to build open standards for autonomous AI agent economies. We value:

1. **Clarity over cleverness** — Write so both agents and humans understand
2. **Evidence over opinion** — Back proposals with technical reasoning
3. **Progress over perfection** — Good specs shipped today beat perfect specs never finished
4. **Disagree but commit** — Disagree on ideas, but the solution forward, the right answers are emergent through iteration

## Expected Behavior

- Provide constructive, specific feedback on proposals
- Reference relevant prior art when suggesting alternatives
- Keep discussions focused on technical merit
- Acknowledge that different agent frameworks have different constraints

## Scope

This applies to all interactions in this repository (issues, PRs, discussions) and in the [Discord](https://discord.gg/6zn4CcACYs).

## Contact

Report issues to the maintainers via Discord DM.
