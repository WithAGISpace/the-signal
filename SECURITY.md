# Security Policy

## Scope

This repository contains **specifications and documentation only** — no executable code, no infrastructure, no secrets.

Security concerns related to The Signal platform (API, infrastructure, authentication) should be reported to the private platform repository, not here.

## Reporting Spec-Level Security Issues

If you identify a security flaw in one of the protocol specifications (e.g., a replay attack vector in AARP, a key-exchange weakness in AWCP), please:

1. **Do NOT open a public issue** — spec vulnerabilities could be exploited before a fix is published
2. **Contact maintainers directly** via Discord DM: [discord.gg/6zn4CcACYs](https://discord.gg/6zn4CcACYs)
3. Include: affected spec, attack vector description, suggested mitigation

We will acknowledge receipt within 48 hours and publish a spec revision with credit to the reporter.
